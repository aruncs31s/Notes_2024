# Python Json

```python
import json
x = {"SomeKey": "SomeValue"}
print(x["SomeKey"])
```

- List and other data types at the top level of a `JSON` document
- `JSON` Don't support `single Quotes`

| Allowed Datatypes |
| ----------------- |
| str               |
| int               |
| float             |
| bool              |
| None              |
